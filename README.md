# LabMate AI
**您的全方位智慧科研協作夥伴**

LabMate AI 是一款專為科研人員、學生及教育工作者設計的全面性實驗管理與協作工具。它結合了嚴謹的資料結構保存方式與強大的真實大型語言模型 (LLM) 輔助功能，幫助您輕鬆追蹤實驗進度、管理各項變數，並迅速獲得關於實驗設計的相關專業建議。

---

## 核心功能

### 1. 實驗數據管理
* **版本控制**：自動替每一次的數據更新紀錄版本號，確保實驗數據的完整可追溯性。
* **結構化儲存**：使用 SQLite 資料庫安全地儲存包含實驗名稱、描述、建立者以及詳細實驗數據等資訊。
* **歷史追蹤**：具備回顧以往歷史數據及修復版本的能力（框架已預留）。

### 2. 實驗計畫協作
* **建構計畫**：使用者可自定義研究目標，並透過建立步驟清單及所需材料來完善計畫。
* **多人協作**：支援新增或移除協作者功能，強化團隊間的合作效率。
* **動態調整**：團隊能隨時根據實驗的變化，彈性修改正在進行中的實驗步驟與內容。

### 3. 真實 AI 科研助理
* **自然語言應答**：本系統已全面整合 OpenAI 相容的 API 端點。除了特定的指令外，您可以直接用日常的自然語言與 AI 對話，請它解釋專有名詞或提供研究建議。
* **智能流程設計**：只要輸入您的研究目標（例如：「研究植物光合作用」），AI 即可依據其豐富的知識庫為您自動生成一套建議的標準實驗操作流程。
* **問題拆解**：將複雜龐大的研究命題，智能拆解為數個可具體執行的子計畫。
* **標準作業程序 (SOP)**：將實驗細項提供給 AI，快速產出包含「操作步驟」與「注意事項」的嚴謹 SOP 文件。

### 4. 儀器使用指南
* **智慧查詢**：快速查找實驗室對應的儀器文檔與使用手冊。
* **高擴充性**：支援使用嚴格的 JSON 格式定義，讓您能自由自訂儀器資料庫。

## 快速開始

### 系統需求
* Python 3.6 或以上版本
* OpenAI 套件 (用於串接 AI 推論核心)
* Streamlit (用於開啟圖形化使用者介面)
* 內建 SQLite3 支援 (Python 標準庫已包含)

### 安裝步驟

1. **下載專案**
   將專案檔案下載或 Clone 至您的本地端電腦中。

2. **環境設定**
   為了能順利運行我們所提供的網頁圖形化介面及使用強大的 AI 對話能力，請先執行以下指令安裝必備的套件：
   ```bash
   pip install streamlit openai
   ```

### 執行說明

**選項一：圖形化使用者介面 (推薦使用)**
使用 Streamlit 執行應用程式，將可開啟現代化且極具互動性的網頁介面。
```bash
streamlit run app.py
```
此指令會啟動一個本地端網頁伺服器（通常位於 `http://localhost:8501`），您可以直接從您的瀏覽器進行操作。

**選項二：終端機自動/互動展示介面**
若要在終端機環境下觀看自動化展示或進行文字互動：
```bash
python "LabMate AI.py" demo
python "LabMate AI.py"
```

## 使用教學 (透過 Web UI 操作)

### 1. 使用者身分設定
當您啟動 Streamlit 應用程式後，請先於左側的側邊欄位中輸入您的姓名或帳號識別碼。此後所有在系統內紀錄的動作，都將會標記為該身分。

### 2. 實驗管理
導航至「實驗數據管理」標籤頁。您可以檢視已註冊的實驗、確認參數，或是建立一個全新的追蹤紀錄。關於數據更新，可直接透過介面提供的 JSON 格式輸入框進行提交。

### 3. 計畫協作
請切換至「實驗計畫協作」標籤頁。您可以在此建立宏觀的研究計畫目標並邀請團隊成員加入。另外，您也能替計畫定調實驗執行所需的標準序列步驟。

### 4. 調用原生 AI 系統
請前往「AI 科研助理」區域體驗高階語言模型的力量！您可以**直接輸入任何問題**請教 AI，或者是使用下方幾個內建的捷徑指令來觸發特殊功能：
* **專屬儀器：** `儀器使用 [儀器名稱]` (搜尋本地 `instrument_docs`)
* **規劃實驗：** `設計實驗流程 [研究目標]`
* **拆解問題：** `拆分問題 [龐大研究命題]`
* **生成規範：** `標準流程 [實驗細節]`

## 進階設定：新增儀器文檔

若您希望擴充儀器查詢系統，請在專案根目錄下建立名為 `instrument_docs` 的資料夾，將 JSON 格式的儀器定義檔案放置於此。

**檔案結構範例：** `instrument_docs/spectrometer.json`

```json
{
    "instrument_id": "inst_002",
    "name": "光譜儀",
    "description": "測量電磁波譜特定部分之光特性的精密儀器。",
    "guide": "spectrometer_guide.txt"
}
```
*請務必於同目錄下建立對應的文字手冊檔案。*
