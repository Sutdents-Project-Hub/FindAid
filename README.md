# LabMate AI
**您的全方位智慧科研協作夥伴**

LabMate AI 是一款專為科研人員、學生及教育工作者設計的全面性實驗管理與協作工具。它結合了嚴謹的資料結構保存方式與強大的真實大型語言模型 (LLM) 輔助功能，幫助您輕鬆追蹤實驗進度、管理各項變數，並迅速獲得關於實驗設計的相關專業建議。

---

## 核心功能

### 1. 實驗數據管理
* **版本控制**：自動替每一次的數據更新紀錄版本號，確保實驗數據的完整可追溯性。
* **結構化儲存**：使用 SQLite 資料庫安全地儲存包含實驗名稱、描述、建立者以及詳細實驗數據等資訊。
* **歷史追蹤**：具備回顧以往歷史數據及修復版本的能力。

### 2. 實驗計畫協作
* **建構計畫**：使用者可自定義研究目標，並透過建立步驟清單及所需材料來完善計畫。
* **多人協作**：支援新增或移除協作者功能，強化團隊間的合作效率。
* **動態調整**：團隊能隨時根據實驗的變化，彈性修改正在進行中的實驗步驟與內容。
* **AI 智慧審核**：點擊按鈕即可讓 AI 從步驟完整性、安全風險、材料充足性與效率優化等角度審核您的計畫，並提供具體改善建議。
* **一鍵採納**：審核完成後可一鍵採納 AI 建議的完整步驟列表，快速優化計畫。

### 3. AI 科研助理（核心模組）
* **自然語言應答**：本系統已全面整合 OpenAI 相容的 API 端點。除了特定指令外，您可以直接用日常的自然語言與 AI 對話，請它解釋專有名詞或提供研究建議。
* **對話自動建立實驗計畫**：透過自然語言（例如「幫我建立一個研究光合作用的實驗計畫」），AI 會自動產生計畫名稱、研究目標、完整步驟與材料清單，並直接寫入系統資料庫。
* **智能流程設計**：輸入研究目標即可自動生成一套建議的實驗操作流程。
* **問題拆解**：將複雜龐大的研究命題，智能拆解為數個可具體執行的子計畫。
* **標準作業程序 (SOP)**：將實驗細項提供給 AI，快速產出包含「操作步驟」與「注意事項」的嚴謹 SOP 文件。

### 4. 文獻與研究建議
* **文獻探索**：輸入研究關鍵字，AI 分析該領域的研究趨勢、推薦學術資料庫與搜尋策略、列出經典文獻方向及頂尖期刊。
* **研究方向建議**：描述現有研究背景或實驗結果，AI 推薦具體可行的後續方向、評估潛在價值並指出可能的突破口。

### 5. 儀器使用指南
* **智慧查詢**：快速查找實驗室對應的儀器文檔與使用手冊。
* **高擴充性**：支援使用嚴格的 JSON 格式定義，讓您能自由自訂儀器資料庫。

## 快速開始

### 系統需求
* Python 3.6 或以上版本
* OpenAI 套件（用於串接 AI 推論核心）
* Streamlit（用於開啟圖形化使用者介面）
* 內建 SQLite3 支援（Python 標準庫已包含）

### 安裝步驟

1. **下載專案**
   將專案檔案下載或 Clone 至您的本地端電腦中：
   ```bash
   git clone https://github.com/Felix-0901/LabMate.git
   ```

2. **環境設定**
   安裝必備的 Python 套件：
   ```bash
   pip install streamlit openai
   ```

### 執行說明

**圖形化使用者介面（推薦使用）**
```bash
streamlit run app.py
```
此指令會啟動一個本地端網頁伺服器（通常位於 `http://localhost:8501`），您可以直接從瀏覽器操作所有功能。

**終端機互動模式**
```bash
python "LabMate AI.py"
```

## 使用教學

### 1. 使用者身分設定
啟動應用程式後，在左側側邊欄輸入您的姓名或帳號識別碼。此後系統內所有操作皆會標記為該身分。

### 2. 實驗數據管理
導航至「實驗數據管理」標籤頁，可以查看已建立的實驗、檢視版本歷史，或建立新的追蹤紀錄。數據更新時系統自動遞增版本號。

### 3. 實驗計畫協作
在「實驗計畫協作」中建立計畫、邀請團隊成員，並逐步新增實驗步驟。每個計畫都支援：
* **AI 智慧審核**：讓 AI 從多個角度檢視計畫的完整性與安全性
* **一鍵採納建議**：直接套用 AI 推薦的改良步驟

### 4. AI 科研助理
前往「AI 科研助理」體驗高階語言模型的力量。支援的指令包括：

| 指令格式 | 功能說明 |
|---|---|
| `幫我建立一個...的實驗計畫` | AI 自動產生完整計畫並寫入系統 |
| `設計實驗流程 [研究目標]` | AI 生成詳細實驗步驟 |
| `拆分問題 [研究命題]` | AI 拆解為可執行的子問題 |
| `標準流程 [實驗細節]` | AI 撰寫 SOP |
| `儀器使用 [儀器名稱]` | 查詢儀器手冊 |
| `文獻 [關鍵字]` | 取得文獻探索建議 |
| `研究方向 [描述]` | 取得研究方向建議 |
| 任意文字 | AI 自由回答科研問題 |

### 5. 文獻與研究建議
在專屬頁面中輸入關鍵字或研究背景，由 AI 提供文獻趨勢分析與後續研究方向建議。

### 6. 儀器使用指南
透過關鍵字搜尋查找實驗室儀器的操作指南。新增儀器請參照 `instrument_docs/` 目錄下的 JSON 格式。

## 進階設定：新增儀器文檔

在專案根目錄下建立 `instrument_docs` 資料夾，放入 JSON 格式的儀器定義檔案。

**檔案結構範例：** `instrument_docs/spectrometer.json`

```json
{
    "instrument_id": "inst_002",
    "name": "光譜儀",
    "description": "測量電磁波譜特定部分之光特性的精密儀器。",
    "guide": "spectrometer_guide.txt"
}
```
